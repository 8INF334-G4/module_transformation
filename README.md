## Dependency

The project uses the following dependencies:
- Java 21: this project has been tested with Java 21. Previous versions may not work.
- Maven: dependency management and build automation

## Compilation

Compilation is handled by maven. To compile the project, run the following command in the terminal:

```bash
mvn clean package
```

## Execution

To run the project, use the following command in the terminal:

```bash
java -jar target/mdt-1.0-SNAPSHOT.jar
```
